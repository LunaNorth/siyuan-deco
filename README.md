# 轻饰笔记

通过自定义属性为思源笔记块添加精美卡片样式，支持图标、标题与多种预设风格，让笔记视觉更统一、更有美感。

## 特别感谢

感谢[QYL主题作者](https://github.com/QYLexpired/QYL-theme) 好多都是借鉴大佬的代码片段


## 使用
安装完插件后 ，打开菜单即可看到轻饰笔记这个选项 ，根据里面的内容即可去使用

<hr>
✨ 让每一块笔记，都拥有属于自己的样子。
